# ED_Lista03.06
## Série Recursiva

Este desafio consiste em criar uma aplicação, em Java, que receba um número natural N e apresente a soma da seguinte série:

```
                    S = 1 + 1/2! + 1/3! + ... + 1/N!
```

```
Requisitos do projeto:

1. Arquitetura MVC;
2. A sequência deve ser realizada por uma função recursiva;
3. Declarar, via comentário, a condição de parada da função.
```
